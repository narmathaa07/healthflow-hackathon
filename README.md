##  Overview

HealthFlow addresses critical resource management challenges in Malaysian hospitals by providing AI-powered predictions for:

- **Bed occupancy forecasting**
- **ICU capacity planning** 
- **Staff allocation optimization**
- **Emergency scenario simulation**

##  Problem Statement

Malaysian hospitals face:
- **Unpredictable patient surges** leading to bed shortages
- **Inefficient staff allocation** during peak periods
- **Limited early warning systems** for capacity crises
- **Manual resource planning** causing delayed responses

##  Our Solution

HealthFlow uses **AI forecasting** to predict hospital resource needs 7 days in advance:

### Key Features:
- **7-Day Bed Occupancy Predictions**
- **Real-time Capacity Alerts**
- **Staffing Requirement Forecasts**
- **Multi-Hospital Scenario Planning**
- **Mobile-First Dashboard Design**

##  Technical Implementation

### AI & Analytics:
- **Time-series forecasting** for bed occupancy
- **Anomaly detection** for crisis alerts
- **Multi-scenario simulation** (Normal, Weekend Surge, Emergency Crisis)
- **Real-time KPI monitoring**

### Tech Stack:
- **Frontend**: Streamlit
- **Visualization**: Plotly
- **Data Processing**: Pandas, NumPy
- **Deployment**: Streamlit Cloud
